# 04_automation

This folder is where I start moving from basic Python scripts into real automation.

Up to this point, I’ve worked with file I/O, loops, conditionals, and building CLI tools. Now the goal is to actually interact with the system and automate real tasks.

The focus here is:
- Running system commands (like ping, ipconfig, etc.)
- Working with files and folders at scale
- Automating repetitive tasks
- Capturing and using command output inside Python
- Building scripts that feel closer to real-world tools

This is where Python starts becoming useful for things like network checks, system monitoring, and general automation.

Each script in this folder is meant to build toward that, not just small exercises but things that could realistically be used or expanded into bigger tools later.

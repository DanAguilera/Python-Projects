02 – Data Structures

This folder focuses on strengthening my understanding of Python data structures, specifically lists and dictionaries.

The goal here is not to build large applications, but to become fluent in how data is stored, accessed, modified, and iterated over. I am intentionally working through smaller, focused exercises before layering in more complex architecture.

Concepts covered in this section include:

Creating and modifying lists

Indexing and slicing

Iterating using for loops and range(len(...))

Creating and updating dictionaries

Looping through .items()

Nested data structures (lists of dictionaries)

Mutability and scope behavior

Mini Project: Pokémon Team Analyzer (Phase 1)

As a practical application of these concepts, I built a small CLI-based Pokémon team manager that:

Stores Pokémon as dictionaries inside a list

Allows adding new Pokémon

Displays team data in structured format

Reinforces real-world data modeling concepts

This section is focused on internalizing how Python handles structured data so that future projects can be built with confidence and clarity rather than guesswork.

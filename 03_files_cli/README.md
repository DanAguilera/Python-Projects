03 – Files & CLI

This section focuses on working with files and building command-line tools that interact with real data.

The goal is to move from in-memory data (lists and dictionaries) to persistent data stored in files, and to build CLI programs that behave like real utilities.

Concepts covered in this section include:

Reading from files (open, read, readlines)

Writing to files (write, append)

Working with structured data in files (JSON)

Understanding file paths and file modes

Separating program logic from stored data

Building CLI tools that manage external data

Handling errors when files do not exist

This section bridges the gap between learning Python fundamentals and building real-world automation tools.

Mini Project: CLI File Manager Tool

As a practical application, this section will include building a CLI-based file manager that can:

Create and write to files

Read and display file contents

Append new data to existing files

Search for specific content

Organize and manage simple data workflows

This section prepares for more advanced topics such as automation, scripting, and system-level programming.
